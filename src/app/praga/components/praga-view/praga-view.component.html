Combate às pragas:

  • Use pesticidas naturais, como óleo de neem, para combater pragas de jardim.
  • Manter a higiene: Limpeza regular para eliminar atrativos e esconderijos das pragas.
  • Plante flores atrativas para polinizadores, como lavanda, para controlar pragas de forma natural.
